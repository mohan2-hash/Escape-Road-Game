# 🚗 Escape Road Game

An interactive browser-based arcade game where players must navigate through busy roads, dodge traffic, and survive as long as possible to achieve the highest score.

## 🧠 Game Concept

Escape Road is an endless driving game inspired by arcade-style mechanics, where the player controls a vehicle and must avoid collisions with other cars, police vehicles, and obstacles while the difficulty gradually increases.

---

## ✨ Features

* 🎯 Smooth player movement (left/right controls)
* 🚗 Dynamic traffic system
* 🚓 Police and obstacle elements
* 📈 Score tracking system
* ⏳ Loading screen with progress bar
* 💥 Game over system with restart
* 🔊 Sound controls (if enabled)
* 🎮 Simple and intuitive UI

---

## 🎯 Controls

* ⬅️ Move Left: `A` or `←`
* ➡️ Move Right: `D` or `→`
* ⏸ Pause: Button on screen
* 🔊 Sound Toggle: Button on screen

---

## ⚙️ How It Works

* The game runs on a continuous loop using `requestAnimationFrame`
* Vehicles spawn dynamically at intervals
* Collision detection determines game over
* Score increases based on survival time
* UI updates in real-time based on player actions
