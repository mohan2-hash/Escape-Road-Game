# 🚗 Escape Road Game

A browser-based arcade game where the player navigates through traffic and avoids collisions.

## 🎮 Features
- Keyboard controls (A/D or Arrow keys)
- Dynamic traffic system
- Score tracking
- Smooth animations

## 🚀 Live Demo
https://comfy-bombolone-adebbf.netlify.app/

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript (Canvas)
